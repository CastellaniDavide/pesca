# Description
This is the pesca for cpp files.

## Required
 - a cpp compiler (eg. gcc for Linux)
 

## Directories structure
 - .github
   - ISSUE_TEMPLATE
     - bug_report.md
     - feature-request.md
 - bin
	 - pesca.cpp
 - doc
   - LICENSE
   - README.md
 - log
	 - trace.log
   
### Execution examples
 - gcc pesca.cpp -o pesca && ./pesca
